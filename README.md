### WIKI

封装了接口调用代理直接访问HTTP请求功能。

### 注意事项
如果是使用war包部署项目的话，推荐使用v1.1.0之后版本；如果是采用jar包部署项目的话，请使用v1.1.3之后的版本；如果想要配合Spring一起使用，现在可以使用v2.0.0版本了，自动化配置。

版本发行：

v1版本将是独立运行版本。

v2版本将配和v1版本对Spring做支持，与哪个版本配套见说明。
